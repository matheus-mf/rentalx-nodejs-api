## Iniciar projeto via Docker

### Criar e iniciar configurações

`docker-compose up`

### Iniciar serviços

`docker-compose start`

### logs

`docker-compose logs rentx -f`

### Para serviços

`docker-compose stop`

### Para e remover as configurações

`docker-compose down`
